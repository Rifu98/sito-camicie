# Backend - Sito Camicie

Skeleton Spring Boot.

Comandi:
- Avviare in locale: mvn spring-boot:run
- Build: mvn -DskipTests package

Config:
- File di configurazione: `src/main/resources/application.properties`
- Database: PostgreSQL (configurazione di default commentata in `application.properties`)

Dipendenze previste: Spring Web, Spring Data JPA, PostgreSQL, Lombok
